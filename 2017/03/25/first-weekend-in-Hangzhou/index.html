first weekend in Hangzhou

在杭州的第一个周末

来杭州工作几天之后，终于迎来了全国人民欢欣鼓舞的周末。周末嘛，当然是要出去走走，哦，不，要去办事情。

由于入职需要去了浙大紫荆港校区办理银行卡，于是在他们的校园内走了走，果然三墩人民公园还是很多人去游玩和休息的。感觉浙大学生太爽了，作为工大学生的我感受到了深深的悲伤。

接下来谈谈对杭州的几点感受

  • 湖滨的人居多，银泰城里的每个餐厅都有大量的人（感觉比北京的多）在排队，这个很让我吃惊，不知道是杭州的设施太少了，还是消费能力太强了，还是有别的原因。
  • 果然银行卡的限制开始影响到我这个无产阶级了，我不知道自己居然有两张农业银行的卡，惊呆我了，以前的卡我记得去银行处理过，难道当时的处理方式是”不要理他，反正没差“嘛。
  • 西湖的喷泉太坑爹，看了一半感觉不知道让我感觉棒棒哒的景象在哪儿。
  • 总有一种感觉，不确定对不对。我总是感觉杭州的矿泉水的价格比北京要高，但是其他价格比北京便宜，不知道是不是成本的原因，按理说在杭州人这么多，成本不会比北京高啊。
  • 一个城市没有地铁是鸡肋，一个人冷风中等地铁这事真不太好受，还有等了好久发现尼玛这车晚上7点就没有了，当时的心里os是：*。
