将HBase导入到eclipse

最近想开始看hbase的源代码，所以想把eclipse导入到eclipse，方便跳转阅读代码。

  • 下载源代码
  • 执行mvn clean install -DskipTests和mvn eclipse:eclipse
  • 执行过程中会出现“错误: 读取/Users/xxx/.m2/repository/org/slf4j/slf4j-log4j12/1.6.4/slf4j-log4j12-1.6.4.jar时出错; zip file is empty”问题，只要把那个文件删除然后重新执行就可以了，不一会儿你就看到build sucess
  • 然后倒入到eclipse中，各种叹号和红叉，崩溃ing。
  • 进入方法发现直接说到了jar，没法查看；转到idea，看到idea直接倒入，并且原来的提示的jar包也可以查看代码。
  • 最后得出结论，何必用eclipse呢，直接用idea不久完了。TODO eclipse导入的mvn问题
