some thinkings about going back home

​ 这次周末回家，我感受最深的dy的服务人员的态度，那态度之差让我有点觉得不是那么自在。这也让我想到了几个问题。

​ 首先第一个问题，dy到底有没有前途了。最近的银泰一直没有开业。之前的金天地变成第一百货，然后就这个样子了。之前可以引以为傲的商场关了。感觉从高中开始，dy一直很稳定的让人失望中。在过年的时候，本人很好奇地查阅了东阳的一些数据，包括人口等，反正很失望（当时还查了杭州的人口增长，居然没有北京快，神他妈尴尬）。现在居然感受到了dy服务如此落后，我真是不知道怎么表达心里那尽情狂奔的万头草泥马。不过这个事情也让我坚定以后要移出dy，也许在今年，因为今年hz落户政策貌似要改变。

​ 最近还想了一个问题，其实很多问题的原因是目前很多人还不具备用脚投票的能力。很多人呆在一个地方，感受到了不好的服务，没心没力去解决问题但是也没有能力去离开那个地方。不能细讲这个问题了。

​ 最后一个问题，本人对未来保持十分悲观地态度。从过年开始的一些感受和听闻只让我觉得未来中国的高端人才是可以和世界最顶尖的人相互pk和谈笑风生。但是剩下的80%甚至90%的人，我真的觉得他们的未来生活会很痛苦。
